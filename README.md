# Python and AI/ML Learning Journey

Long-term engineering path from Python fundamentals to industrial AI, robotics,
computer vision, and research-oriented AI engineering.

## Goal

Build a strong Python, data, and ML foundation during the first six months. The
long-term target is AI Industrial Architect / Research Engineer.

## Track

`Python and Git` -> `Data and SQL` -> `Classical ML` -> `Computer Vision` ->
`MLOps` -> `Final Project`

## Current Status

Month 1 is complete. The covered topics include Python fundamentals, functions,
collections, exceptions, file handling, debugging, `pathlib`, Pillow basics,
introductory OOP, Git branches, pull requests, and systematic code review.

The Month 1 assessment and targeted retest are complete. The next stage is
Month 2: NumPy, pandas, SQL, and exploratory data analysis (EDA).

## Projects

- `projects/log_analyzer` - CNC log analyzer with a report and project README.
- `month1/week2/calculator.py` - early calculator exercise being hardened during
  the Month 1 review.

## Structure

- `month1/`, `month2/`, ... - lesson files organized by month and week.
- `projects/` - mini-projects and portfolio work.
- `notes/` - durable topic summaries and terminology.
- `docs/` - documentation map plus archived planning baselines.
- `state/learning_state.yaml` - compact current checkpoint.

Local mentor documents (`PROGRESS.md`, `CLAUDE.md`, and the frozen
`ROADMAP_RU.md`) are intentionally excluded from the public repository.

## Stack

- Python 3.13
- Git and GitHub
- VS Code
- Pillow
- NumPy, pandas, scikit-learn, OpenCV, FastAPI, Docker, and MLflow (planned)
