# Month 1 Plan

> Status: archived initial plan. It records the original intent but does not
> define the current lesson order. Use local `PROGRESS.md` for operational truth.

## Objective

Build the first operational Python layer and remove syntax fear.

## Week 1

- variables;
- base types: `int`, `float`, `bool`, `str`;
- operators;
- expressions;
- formatted output;
- predicting code behavior before execution.

Output:

- simple engineering calculations in Python.

## Week 2

- conditions;
- comparison operators;
- boolean logic;
- loops;
- function basics.

Output:

- small decision scripts and repeated calculations.

## Week 3

- lists;
- dictionaries;
- indexing;
- iteration over collections;
- reading and writing text files;
- basic CSV handling.

Output:

- simple log and measurement parsing.

## Week 4

- functions with parameters and returns;
- code organization;
- basic modules;
- error patterns;
- introduction to Git.

Output:

- first structured mini-project with version control.

## Monthly Deliverables

- `4+` completed lessons;
- `20+` short exercises;
- `1` engineering mini-project;
- `1` initialized Git repository;
- updated progress log and learning state.
