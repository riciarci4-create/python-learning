# Documentation Map

This directory contains public reference documents. It is not the source of the
current lesson checkpoint.

## Sources of Truth

1. Local `PROGRESS.md` - actual lessons, notes, debts, exams, and next action.
2. `state/learning_state.yaml` - compact machine-readable checkpoint.
3. Local `CLAUDE.md` - mandatory mentor behavior.
4. Local frozen `ROADMAP_RU.md` - the 7-10 year strategic direction.

## Files in This Directory

- `command_protocol.md` - command meanings and lesson interaction protocol.
- `evidence_based_learning.md` - active learning mechanics and mastery criteria.
- `learning_system.md` - archived initial learning-system baseline.
- `month_01_plan.md` - archived initial Month 1 plan.
- `year_01_roadmap.md` - archived initial Year 1 outline.

Archived documents preserve planning history. They must not override the current
lesson order in `PROGRESS.md`.
