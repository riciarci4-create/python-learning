## Цель

Использовать цикл, число повторений которого заранее неизвестно, и обеспечивать
условие его завершения.

## Теория

`while` повторяет блок, пока условие истинно. Перед каждой итерацией условие
проверяется заново.

```python
attempt = 0

while attempt < 3:
    print(attempt)
    attempt += 1
```

Если переменные условия никогда не меняются, цикл может стать бесконечным.
`number += 1` является сокращением для `number = number + 1`: сначала вычисляется
правая часть, затем новое значение присваивается переменной.

## Когда выбирать цикл

- `for` — известна коллекция или диапазон;
- `while` — повторять до события или изменения состояния.

## Микропримеры

```python
while machine_status != "ready":
    machine_status = read_status()
```

```python
count = 5
while count > 0:
    count -= 1
```

Бесконечную программу в терминале обычно останавливают `Ctrl + C`.

## Инженерный смысл

`while` применяется в ожидании готовности оборудования, повторных попытках и
длительно работающих сервисах. В реальной системе дополнительно нужны тайм-аут,
лимит попыток и обработка аварийного завершения.

## Что было понято и исправлено

- `while` сам не изменяет состояние.
- Пропущенный `number += 1` создал бесконечный цикл.
- Для безопасного цикла нужно назвать условие выхода до написания тела.

## Math Core

Изменение счётчика задаёт рекуррентную последовательность: `n_next = n + 1`.
Завершение возможно, если каждое обновление приближает состояние к границе.

## English Core

- **while loop** — цикл с условием;
- **counter** — счётчик;
- **condition** — условие;
- **infinite loop** — бесконечный цикл;
- **termination** — завершение;
- **attempt** — попытка.
