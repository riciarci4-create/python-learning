## Цель

Безопасно читать текстовый файл построчно, считать события и сохранять итоговый
отчёт как отдельный артефакт.

## Теория открытия файла

```python
with open(log_path, "r", encoding="utf-8") as log_file:
    for log_line in log_file:
        ...
```

Контекстный менеджер `with` закрывает файл после выхода из блока, даже если
возникло исключение. Явная `encoding="utf-8"` делает работу с текстом
воспроизводимее между системами.

`read()` загружает всё содержимое сразу. Цикл по файлу обрабатывает строки по
одной и подходит для больших логов.

## Подсчёт и фильтрация

```python
line_count = 0

for log_line in log_file:
    line_count += 1
    if "CNC" in log_line:
        cnc_count += 1
```

Широкий признак `"CNC"` считает категорию, точный `"Program running"` — конкретное
событие. Условие должно соответствовать смыслу метрики.

## Запись отчёта

```python
with open(report_path, "w", encoding="utf-8") as report_file:
    report_file.write(report_text)
```

Режим `w` создаёт файл или полностью перезаписывает существующий. `write()` не
добавляет перенос строки автоматически; `\n` нужно включить в текст.

## Инженерный смысл

Файл отчёта — воспроизводимый артефакт: его можно передать, сравнить между
запусками, включить в pipeline. Итог формируется после завершения цикла, когда
счётчики уже накоплены полностью.

## Math Core

- Счётчик является дискретной суммой индикаторов совпадения.
- Строки разбиваются на подмножества по признакам.
- Инвариант: `0 <= event_count <= line_count`.

## English Core

- **file** — файл;
- **read / write** — читать / записывать;
- **line** — строка файла;
- **count** — счётчик или количество;
- **report** — отчёт;
- **encoding** — кодировка;
- **not found** — не найден.

## Anki

1. **Чем `print()` отличается от `write()`?** — `print` по умолчанию добавляет перевод строки в терминале, `write` записывает только переданный текст.
2. **Почему отчёт собирают после цикла?** — Только тогда счётчики содержат итог по всем строкам.
3. **Что произойдёт при чтении отсутствующего файла?** — Возникнет `FileNotFoundError`.
4. **Когда проверять точную фразу?** — Когда широкая подстрока может дать ложные срабатывания.
