## Цель

Хранить упорядоченную последовательность и выполнять одно действие для каждого
её элемента.

## Теория

`list` — изменяемая упорядоченная коллекция. Цикл `for` поочерёдно связывает
переменную цикла с каждым элементом коллекции.

```python
machines = ["CNC-1", "CNC-2", "CNC-3"]

for machine in machines:
    print(machine)
```

`range(start, stop)` создаёт последовательность целых чисел, в которой `stop`
не включается. `range(1, 5)` даёт `1, 2, 3, 4`.

## Микропримеры

```python
for number in range(1, 6):
    print(number)
```

```python
for number in range(1, 21):
    if number % 2 == 0:
        print(number)
```

```python
temperatures = [71.2, 72.0]
temperatures.append(72.4)
```

Вложенный `if` получает дополнительный уровень отступа, потому что проверка
выполняется внутри каждой итерации.

## Инженерный смысл

Через `for` обрабатывают строки журнала, показания датчиков, файлы в каталоге и
строки таблицы. Имя коллекции полезно писать во множественном числе, имя одного
элемента — в единственном: `log_lines` и `log_line`.

## Math Core

- `range` задаёт конечную дискретную последовательность.
- Проверка `n % 2 == 0` определяет множество чётных чисел.
- Количество итераций для `range(start, stop)` при шаге 1 равно `stop - start`.

## English Core

- **list** — список;
- **loop** — цикл;
- **iteration** — одна итерация;
- **item / element** — элемент;
- **range** — диапазон;
- **even / odd** — чётный / нечётный.
