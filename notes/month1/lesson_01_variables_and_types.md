## Цель

Понимать, как Python хранит значения, различать текст и числа и выбирать тип,
который соответствует дальнейшим операциям.

## Теория

Переменная — имя, связанное со значением. В выражении `age = 27` оператор `=`
не сравнивает значения, а связывает имя `age` с целым числом `27`.

Основные типы урока:

- `str` — текст в кавычках: `"CNC started"`;
- `int` — целое число: `27`;
- `float` — число с дробной частью: `27.5`.

Тип определяет допустимые операции. Python не складывает строку и число
автоматически: `"27" + 1` вызывает `TypeError`. Преобразование должно быть
явным: `int("27") + 1`.

## Микропримеры

```python
machine_name = "CNC-1"       # str
part_count = 120              # int
temperature = 73.5            # float
```

```python
print(type(part_count))        # <class 'int'>
print(10 // 3)                 # 3
print(10 % 3)                  # 1
```

`//` выполняет целочисленное деление, `%` возвращает остаток. Остаток полезен
для проверки чётности, циклических событий и распределения деталей по группам.

## Инженерный смысл

Неверный тип меняет поведение системы. Строка `"100"` не является количеством
деталей, пока не преобразована и не проверена. В реальном сервисе типы приходят
из файлов, датчиков, API и пользовательского ввода, поэтому границы системы
требуют валидации.

## Что было понято и исправлено

- Текст записывается в кавычках, числа — без кавычек.
- `"27"` и `27` представляют разные данные.
- Возраст сначала был записан как `"27"`; после исправления стал `int`.

## Math Core

- `+`, `-`, `*`, `/` — обычные арифметические операции.
- `a = q * b + r` связывает деление `//` и остаток `%`, где `q = a // b`,
  `r = a % b`.

## English Core

- **variable** — переменная, имя значения;
- **value** — значение;
- **string** — строка текста;
- **integer** — целое число;
- **floating-point number** — число с плавающей точкой;
- **type** — тип данных.
