## Цель

Понимать изображение как прямоугольную сетку пикселей, создавать и открывать
файлы и извлекать базовые свойства.

## Теория

```python
from PIL import Image

image = Image.new("RGB", (200, 100), (255, 0, 0))
image.save("assets/test_image.png")
```

Аргументы `Image.new`:

1. `"RGB"` — цветовой режим;
2. `(200, 100)` — размер `(width, height)`;
3. `(255, 0, 0)` — цвет каждого пикселя.

Открытое изображение предоставляет свойства:

```python
opened_image = Image.open(image_path)
opened_image.format   # PNG
opened_image.size     # (200, 100)
opened_image.mode     # RGB
```

## Координаты и пиксели

Начало координат находится в верхнем левом углу. `x` растёт слева направо, `y`
— сверху вниз. `getpixel((0, 0))` читает верхний левый пиксель.

```python
width, height = opened_image.size
center_x = width // 2
center_y = height // 2
center_pixel = opened_image.getpixel((center_x, center_y))
```

Координаты должны быть целыми, поэтому использовано `//`.

## Инженерный смысл

Для Computer Vision изображение становится числовыми данными: координаты
задают положение, каналы — интенсивности цвета. Позже NumPy представит RGB-
изображение массивом формы `(height, width, channels)` — порядок отличается от
Pillow-свойства `size`.

## Math Core

- Изображение задаёт координатную сетку `width * height` пикселей.
- RGB-пиксель является тройкой чисел.
- Центр вычисляется целочисленным делением координатных размеров.

## English Core

- **image** — изображение;
- **pixel** — пиксель;
- **width / height** — ширина / высота;
- **format** — формат файла;
- **color mode** — цветовой режим;
- **channel** — канал;
- **center** — центр.
