## Цель

Выделять повторяемую операцию в именованный блок, передавать входные данные и
возвращать результат для дальнейшей обработки.

## Теория

Функция определяется через `def`, но выполняется только при вызове.

```python
def calculate_area(width, height):
    return width * height


area = calculate_area(200, 100)
```

`width` и `height` — параметры определения. `200` и `100` — аргументы конкретного
вызова. `return` завершает функцию и передаёт значение вызывающему коду.

`print()` создаёт видимый вывод, но не заменяет `return`. Функция без явного
`return` возвращает `None`.

## Микропримеры

```python
def is_even(number):
    return number % 2 == 0
```

```python
def normalize_status(status):
    return status.strip().lower()
```

```python
raw_count = input("Количество: ")
count = int(raw_count)
```

Функция должна получать всё необходимое через параметры и не зависеть от
случайных внешних переменных.

## Math Core

Программная функция похожа на отображение `f(x)`: вход преобразуется в выход.
Для детерминированной функции одинаковые входы дают одинаковый результат, если
нет зависимости от изменяемого внешнего состояния.

## English Core

- **function** — функция;
- **parameter** — имя входа в определении;
- **argument** — конкретное значение при вызове;
- **return value** — возвращаемое значение;
- **call** — вызов функции;
- **result** — результат.

## Anki

1. **Когда функция выполняется?** — Только при вызове.
2. **Чем параметр отличается от аргумента?** — Параметр находится в определении, аргумент передаётся при вызове.
3. **Чем `print` отличается от `return`?** — `print` выводит, `return` передаёт значение вызывающему коду.
4. **Что возвращает функция без `return`?** — `None`.
