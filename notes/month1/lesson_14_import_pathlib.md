## Цель

Подключать стандартные модули и строить пути, не зависящие от папки запуска.

## Теория импорта

```python
from pathlib import Path
```

`pathlib` — стандартный модуль, `Path` — класс для объектной работы с путями.
`path` в имени переменной обычно является соглашением, а не командой Python.

## Два разных основания пути

```python
Path.cwd()       # папка, из которой запущен процесс
Path(__file__)   # расположение текущего Python-файла
```

`Path("machine_log.txt")` является относительным путём от current working
directory. Он не ищет файл по всему компьютеру.

```python
script_path = Path(__file__)
script_dir = script_path.parent
project_root = script_dir.parent.parent
log_path = project_root / "machine_log.txt"
```

Для объекта `Path` оператор `/` соединяет части пути. `.parent` поднимается на
один уровень, `.exists()` возвращает `True` или `False`.

## Метод и вызов метода

`log_path.exists` — объект метода. `log_path.exists()` — его выполнение и
булев результат. Скобки принципиальны.

## Инженерный смысл

Скрипт может запускаться из терминала, IDE, теста или сервиса. Путь к ресурсу
проекта часто нужно строить от `__file__`, иначе программа «работает только на
моём компьютере» или только из одной директории.

## Что было понято и исправлено

- `Path("name")` не выполняет глобальный поиск.
- `.parent` возвращает папку, а не «родительский файл».
- `exists` без скобок исправлен на `exists()`.
- Различие `cwd` и местоположения файла воспроизведено на месячном экзамене.

## Math Core

Путь — иерархия узлов. Каждый `.parent` выполняет переход на один уровень вверх;
оператор `/` добавляет дочерний компонент.

## English Core

- **module** — модуль;
- **import** — подключение кода;
- **path** — путь;
- **relative / absolute path** — относительный / абсолютный путь;
- **current working directory** — текущая рабочая директория;
- **parent directory** — родительская папка;
- **exists** — существует.

## Anki

1. **Чем `Path.cwd()` отличается от `Path(__file__)`?** — Первое описывает папку запуска, второе — расположение файла скрипта.
2. **Что делает `.parent`?** — Возвращает родительский каталог пути.
3. **Что возвращает `.exists()`?** — `True` или `False`.
4. **Почему путь от `__file__` устойчивее?** — Он не меняется при запуске из другой рабочей директории.
