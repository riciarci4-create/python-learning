# Notes

Use this directory for durable topic summaries that remain useful after a lesson:

- core mental models and transferable patterns;
- terminology with short English-to-Russian explanations;
- common failure modes and debugging checks;
- engineering tradeoffs and links to later ML or industrial applications.

The current checkpoint and lesson debt do not belong here. They remain in local
`PROGRESS.md` and `state/learning_state.yaml`.

## Lesson indexes

- [Month 1 — Python and Git](month1/README.md)

Create one separate durable note for every completed lesson and link it from the
operational `PROGRESS.md`. A lesson note must preserve the full theory,
engineering examples, observed misconceptions, Math Core, English Core, Anki
questions with answers, and mastery criteria.
